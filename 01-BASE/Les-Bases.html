Les Formulaires :

  • La Balise FORM permet de créer le formulaire.
  • Name : Cet attribut est OBLIGATOIRE et permet de renseigner le script / la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du Formulaire. Cette page va recevoir l'ensemble des données saisies par mon utilisateur.

Method : Permet de préciser quelle méthode [GET | POST] doit être utilisé par le navigateur pour transmettre les données saisies par l'utilisateur vers la page ACTION.

















Créer un lien en HTML

  • La balise "a" permet de créer un lien vers une autre ressource. (Images, Vidéos, PDF, ZIP, ...)

  • L'attribut href : Permet de définir le chemin vers le fichier à lier.
  • L'attribut target : Prend plusieurs valeurs : "_blank" pour ouvrir un lien dans une nouvelle fenêtre par exemple...
  • L'attribut title : Permet de donner un titre au lien. Ce titre sera afficher dans une bulle au survol du line par la souris.
